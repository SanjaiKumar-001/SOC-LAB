# SOC-LAB

This repository documents a Microsoft Sentinel SOC lab built through attack simulations, detection engineering, and incident investigations.

Each investigation follows a structured workflow from attack simulation to detection, alert generation, investigation, and documentation, with supporting evidence and Kusto Query Language (KQL) queries.

## Project Goal

Build and document 100 practical SOC investigations covering endpoint security, Active Directory, Microsoft security technologies, network security, attack chains, and independent threat scenarios.

## Lab

The complete lab architecture, environment configuration, and data collection setup are documented in the `00-Environment` directory.
